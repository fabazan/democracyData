Opening a ZIP file:
ZIP files (*.zip) are compressed. In Windows, for instance, 'My Computer' can show and decompress zip files. Alternatively free software tools like 7-zip or WinZip can be used to decompress them.

--
Opening an SPSS Portable file:
Using SPSS, select in drop-down menu File -> Open -> Data, change the file type to SPSS Portable (*.por), select the file name and click on Open. NOTE! If Scandic characters å,ä and ö are not properly displayed (the problem occurs with SPSS 21 in particular): Choose from the drop-down menu Edit -> Options -> General -> Character encoding for data and syntax -> Local. Open the file as instructed above.

--
Html files of qualitative data:
Save and keep all html files in the same folder. Start your browsing from the file daFnnnn_index.html which contains the links to the actual data files.
